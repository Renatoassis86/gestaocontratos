Como instalar no Gmail

  1. Clique em ⎘ Copiar assinatura acima — o HTML da assinatura vai para o clipboard.
  2. Abra o Gmail → clique na engrenagem → Ver todas as configurações.
  3. Role até Assinatura → clique em Criar nova.
  4. No editor de assinatura, clique nos três pontos ⋮ → Editar HTML.
  5. Cole o HTML copiado e clique em Salvar alterações.

Como instalar no Outlook

  1. Copie a assinatura acima clicando no botão.
  2. Abra o Outlook → Arquivo → Opções → Email → Assinaturas.
  3. Clique em Nova, nomeie a assinatura e cole no campo de texto.
  4. Clique em OK para salvar.

Importante

  1. Substitua os e-mails [nome]@arkosintelligence.com pelos e-mails reais de cada fundador.
  2. Atualize o link do LinkedIn com o perfil pessoal de cada um.
  3. Teste enviando um e-mail para si mesmo antes de usar oficialmente.
